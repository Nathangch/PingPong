# 🚀 Neon Ping Pong

Um jogo clássico de **Ping Pong** com estética neon vibrante, desenvolvido em **Python utilizando a biblioteca Pygame**. O projeto oferece uma experiência fluida com recursos modernos de personalização e modos de jogo.

---

## 🎮 Novas Funcionalidades

- **🏠 Menu Inicial Interativo**: Interface completa para navegar entre as opções do jogo.
- **🌈 Personalização de Cores**: Escolha entre diversas cores neon para os seus paddles e pontuações.
- **🤖 Modo Desafio (PvCPU)**: Jogue contra uma Inteligência Artificial balanceada se estiver sozinho.
- **👥 Modo Clássico (PvP)**: Compita contra um amigo no mesmo teclado.

---

## 🕹️ Como Jogar

O jogo começa no **Menu Principal**. Use as setas `↑` e `↓` para navegar e `Enter` para selecionar.

### Menu de Cores
No menu **CHOOSE COLORS**, use `↑` e `↓` para selecionar o jogador e `←` e `→` para trocar a cor. Selecione **BACK** e pressione `Enter` para salvar e voltar.

### Controles na Partida
- **Jogador 1 (Esquerda)**: `W` (Cima) / `S` (Baixo)
- **Jogador 2 (Direita)**: `↑` (Cima) / `↓` (Baixo) *(ou controlado pela CPU)*
- **Sair da Partida**: `Esc` para voltar ao menu.

---

## ✨ Características Técnicas
- **Movimento Fluido**: Sistema de FPS cravado em 60 para máxima suavidade.
- **Efeito Neon Glow**: Efeitos visuais dinâmicos nos paddles e na bola.
- **Velocidade Progressiva**: A bola acelera a cada colisão, aumentando o desafio.
- **Interface Responsiva**: Menus intuitivos com feedback visual.

## 🚀 Como Executar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/Nathangch/PingPong.git
```

2. Certifique-se de ter o Pygame instalado:
```bash
pip install pygame
```

3. Execute o jogo:
```bash
python ping_pong.py
```
